New Indicator

This wizard creates a User Defined Indicator.

Please select the Language and input the SQL Template in the Wizard. These keywords can be use here:
<%=__WHERE_CLAUSE__%>, replace the where clause provided by the data filter;
<%=__TABLE_NAME__%>,replaces the table name;
<%=__COLUMN_NAMES__%>,replaces the column(s);
<%=__GROUP_BY_ALIAS__%>, replaces "group by" whole clause.

When you complete the User Defined Indicator, the Editor will open, the default category of User Defined Indicator is "User Defined Count".
