Analyse de corrélation de colonnes :

Analyse de corrélation numérique

Ce type d'analyse affiche les corrélations entre les données nominales et numériques dans un graphique à bulles. Pour chaque valeur nominale, la moyenne des données numériques est calculée et représentée sur l'axe vertical, et la valeur nominale est représentée sur l'axe horizontal.

L'un des objectifs de cette analyse est d'identifier facilement les problèmes de qualité de données en regardant les valeurs extrêmes dans le graphique.

Analyse de corrélation temporelle

Ce type d'analyse affiche les corrélations entre les données nominales et temporelles dans un graphique de Gantt. Pour chaque valeur nominale, les valeurs minimale et maximale des données temporelles sont calculées et représentées sous forme de barre dans le graphique.

Analyse de corrélation nominale

Ce type d'analyse ne fonctionne qu'avec des valeurs nominales et affiche les relations entre des paires de données dans le graphe de réseau. Il met en relief les relations les plus faibles entre les paires de données afin de trouver de potentielles relations incorrectes.
