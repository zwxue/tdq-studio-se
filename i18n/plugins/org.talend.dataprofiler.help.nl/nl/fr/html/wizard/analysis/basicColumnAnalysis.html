Analyse de colonne :

Analyse simple de colonne

Cet assistant génère une analyse de colonne vide et ouvre l'éditeur d'analyse. Vous pouvez sélectionner les colonnes à analyser et assigner manuellement les indicateurs à chaque colonne, comme le nombre de nulls, la table de fréquence, le résumé statistique, les indicateurs de correspondance de modèle, etc.
